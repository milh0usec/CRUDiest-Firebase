MEAN stack ReSTful CRUD app with Angular front-end and Bootstrap styling.

This website is two apps, with separate back and front ends. The back end uses Node, Express, and MongoDB and is deployed on Heroku. The data is served to an API as JSON objects. MongoDB is accessed via the lightweight, schemaless Monk Node modules.

The front end uses Angular and Bootstrap and is deployed on Firebase. There are three views: INDEX/NEW (Home), SHOW, and EDIT. The asynchronous typeahead reduces the typing required to add a new movie to a few keystrokes so a separate NEW view isn't needed.

The asynchronous typeahead is implemented with UI Boostrap. Each keystroke makes an HTTP request to the Open Movie Database, which provides ten movie titles. onSelect (clicking a movie title) fires another HTTP request to the OMDB which returns the movie data. A final HTTP request triggers $scope.$apply() to refresh the view, adding the new movie to the home page.

The app is fully responsive. The number of movies displayed per row changes from five in the large view, to four in the medium view, three in the small view, and two on mobile devices.

Comments are an array of objects nested in the movie object. Other features include likes/dislikes.

The back end code and front end code are on GitHub. Tutorials to make the back end and the front end are also on GitHub.
