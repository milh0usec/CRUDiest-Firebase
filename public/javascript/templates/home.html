MEAN stack ReSTful CRUD app with Angular front-end and Bootstrap styling.

This website is two apps, with separate back and front ends. The back end uses Node, Express, and MongoDB and is deployed on Heroku. The data is served to an API as JSON objects. MongoDB is accessed via the lightweight, schemaless Monk and MongoSkin Node modules.

The front end uses Angular and Bootstrap and is deployed on Firebase. There are four views: INDEX (Home), NEW, SHOW, and EDIT. Comments are an array of objects nested in the movie object. Other features include likes/dislikes. The app is fully responsive. The number of movies displayed per row changes from five in the large view, to four in the medium view, three in the small view, and two on mobile devices.

The back end code and front end code are on GitHub. Tutorials to make the back end and the front end are also on GitHub.
